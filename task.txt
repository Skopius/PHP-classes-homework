    Простое задание: Создание класса "Калькулятор"
        Создайте класс "Calculator", который будет иметь методы для выполнения простых математических операций: сложение, вычитание, умножение и деление.
        Каждый метод должен принимать два числа в качестве аргументов и возвращать результат операции.

    Среднее задание: Работа с геометрическими фигурами
        Создайте классы для различных геометрических фигур, таких как круг, прямоугольник, треугольник.
        Каждый класс должен иметь методы для вычисления площади и периметра фигуры.
        Предусмотрите валидацию входных данных (например, положительные значения для длин сторон).

    Средне-сложное задание: Управление банковским счетом
        Создайте класс "BankAccount", представляющий банковский счет.
        Класс должен иметь методы для внесения и снятия денег, а также для проверки баланса.
        Реализуйте возможность защиты счета паролем, который нужно будет указывать при снятии денег.

    Сложное задание: Управление задачами
        Создайте класс "Task", представляющий задачу в списке дел.
        Класс должен иметь свойства для описания задачи, статуса (выполнена или нет) и срока выполнения.
        Реализуйте методы для установки статуса задачи, изменения срока выполнения и получения информации о задаче.
        Создайте дополнительный класс "TaskList", который будет хранить список задач и предоставлять методы для добавления, удаления и отображения задач.

    Очень сложное задание: Разработка системы заказов в интернет-магазине
        Создайте классы для моделирования системы заказов в интернет-магазине.
        Реализуйте классы "Product" (товар), "Order" (заказ) и "Customer" (покупатель).
        Класс "Product" должен иметь свойства, описывающие товар, такие как название, цена, количество и т.д.
        Класс "Order" должен представлять собой заказ, который содержит список товаров, сумму заказа, данные о покупателе и т.д.
        Класс "Customer" должен содержать информацию о покупателе, такую как имя, адрес и т.д.
        Реализуйте методы для добавления товаров в корзину, оформления заказа, расчета общей суммы заказа и т.д.
